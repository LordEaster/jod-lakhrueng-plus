# จดละครึ่ง พลัส

เครื่องมือช่วยจดรายการซื้อและสรุปสิทธิของโครงการไทยช่วยไทย พลัส (60/40) แบบเก็บข้อมูลในเครื่องเป็นหลัก

## คุณสมบัติ

- จดรายการซื้อได้รวดเร็ว
- ดูสรุปยอดรายวันและรายเดือน
- ดูประวัติรายการย้อนหลัง
- ตั้งค่าการใช้งานและการแสดงผล
- ส่งออกและนำเข้าข้อมูลได้
- รองรับการใช้งานแบบ PWA

## การพัฒนา

ติดตั้ง dependencies:

```bash
npm install
```

รันโหมดพัฒนา:

```bash
npm run dev
```

สร้างไฟล์สำหรับใช้งานจริง:

```bash
npm run build
```

รันชุดทดสอบ:

```bash
npm run test
```

ตรวจ lint:

```bash
npm run lint
```

## โครงสร้างโปรเจกต์

- `src/pages/` หน้าหลักของแอป
- `src/components/` คอมโพเนนต์ UI ที่ใช้ซ้ำ
- `src/hooks/` logic สำหรับดึงและจัดการสถานะ
- `src/db/` ชั้นข้อมูลที่ใช้กับ Dexie / IndexedDB
- `src/logic/` ฟังก์ชันคำนวณและฟอร์แมตข้อมูล
- `public/` ไฟล์ static เช่นโลโก้และ `CNAME`
- `docs/` เอกสารประกอบโครงการ
